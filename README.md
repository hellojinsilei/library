### 安装完成后即可在控制台运行命令
### 如何使用
```
$ git clone https://github.com/hellojinsilei/library.git

$ cd Books-Management-System

$ mvn clean compile

$ mvn clean package

$ mvn clean install

$ mvn jetty:run

打开浏览区输入http://localhost:9000
```
